# Contributing

**notyourface** is explicitly meant as an Open Source project that is open for community contributions. You are welcome to open Pull Requests with bug fixes or feature implementations as long as you are the sole author of the code you contribute and you agree to publish it as part of the **notyourface** codebase, licensed under the MIT license.
